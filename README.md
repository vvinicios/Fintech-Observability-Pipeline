# Fintech-Observability-Pipeline
End-to-end Data Engineering pipeline focused on financial transactions reliability, using AWS Databricks, Airflow and Great Expectations!! 

PT-BR
🎯 Objetivo do Projeto (The Big Picture)
O objetivo deste projeto é construir uma Plataforma de Dados Resiliente e Observável para processamento de transações financeiras em larga escala.

- Diferente de pipelines comuns que apenas "movem dados", esta plataforma foca no Ciclo de Confiabilidade (Reliability Loop):
- Ingestão Escalável: Simulação de alta volumetria de dados usando Python Generators e Docker.
- Processamento em Camadas (Medallion Architecture): Organização dos dados em Bronze, Silver e Gold dentro de um ambiente Databricks/Spark.
- Observabilidade Nativa: Implementação de um loop de qualidade de dados (Data Quality) que monitora falhas, anomalias e integridade em tempo real, gravando metadados de execução para auditoria.
- Orquestração de Classe Enterprise: Gerenciamento de workflows complexos utilizando Apache Airflow rodando em Kubernetes.
